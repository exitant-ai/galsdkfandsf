# 무한순환 매매 시뮬레이터 봇 (TQQQ / QLD / SOXL)

하락 마감일에만 LOC로 매수하는 무한순환 매매법을, **실제 과거 데이터**로 매일 따라가며 체험하는 텔레그램 채팅 스타일 시뮬레이터입니다. 자기 전 LOC 세팅 → 다음날 결과 확인 → 익절·보관·투입·쿼터까지, 실전 흐름을 그대로 미리 연습할 수 있습니다.

## 특징

- **종목 3개**: TQQQ(나스닥100 3배) / QLD(나스닥100 2배) / SOXL(반도체 3배) — 2010~2026 실데이터 내장
- **하락일 전용 매수** (1~10%) · 매수가마다 **+N% GTC 익절**
- **보관(금고)·투입·쿼터(현금 소진 시 1/4 매도)** 자동 처리 + 안내
- **오후 7시 LOC 세팅 알림** + 미완료 시 리마인더
- **진행상황 자동 저장**(브라우저) → 껐다 켜도 이어서
- **현황 보기 / 직접 수정**(실제 계좌와 맞추기)

> ⚠️ 이 도구는 매매 보조·시뮬레이션용이며 투자 자문이 아닙니다. 투자 판단과 책임은 본인에게 있습니다.

## 그냥 쓰는 법 (제일 간단)

`index.html` 파일을 **브라우저로 더블클릭**하면 끝. 설치·서버 필요 없습니다. 모든 데이터와 코드가 이 파일 하나에 들어 있어요.

## 깃허브에 올려서 인터넷 주소로 쓰는 법 (GitHub Pages)

1. 깃허브에서 **New repository** 생성 (예: `tqqq-sim`) → Public
2. 이 폴더의 `index.html` (와 `README.md`)을 **Add file → Upload files**로 업로드 → Commit
3. 저장소 **Settings → Pages**
4. **Source: Deploy from a branch** → Branch: `main` / 폴더: `/ (root)` → **Save**
5. 1~2분 뒤 `https://<내아이디>.github.io/tqqq-sim/` 주소로 접속하면 어디서든 사용 가능

휴대폰 브라우저에서도 똑같이 열려요. 홈 화면에 추가하면 앱처럼 쓸 수 있습니다.

## 올릴 파일

이 폴더 안의 것만 올리면 됩니다:

```
index.html      ← 시뮬레이터 본체 (이거 하나면 동작)
README.md       ← 설명서 (선택)
.gitignore      ← 업로드 제외 목록 (선택)
```

`_sim_template*`, `_t3`, `_check`, `_QQQ.json` 등 **밑줄(_)로 시작하는 파일은 전부 작업용 임시 파일**이라 올릴 필요 없습니다(삭제해도 됩니다).
